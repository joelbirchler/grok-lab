# Grok Lab

Grok Lab is a learning platform for live demo and experimentation of code.

## Building

Build ClojureScript with:

    $ lein cljsbuild once
    (or)
    $ lein cljsbuild auto


## Running

Fire up the server:

    $ lein ring server
